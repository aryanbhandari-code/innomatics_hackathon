# Innomatics test/Hackathon

## Overview
This project integrates food delivery data from multiple sources (CSV, JSON, and SQL) to create a unified dataset for business analysis.

## Files
* **final_food_delivery_dataset.csv**: The integrated dataset containing order, user, and restaurant information.
* **Innomatics_Test.ipynb**: The Python notebook containing the data cleaning, merging logic, analytical queries of the MCQs and Numeric entry questions.

## Key Features
* **Data Merging **: Combined datasets using `user_id` and `restaurant_id` as join keys.
* **Metric Calculation**: Analyzed total revenue, average order values, and city-wise performance.
* **Segmentation**: Evaluated order patterns specifically for Gold membership users.
* **Every query of question is solved and run in the file, and the answer based on the output is inserted in google form.**

## Tech Stack
* Python
* Pandas
* SQL/SQLite
